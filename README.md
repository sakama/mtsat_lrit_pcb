# MTSAT LRIT PCB

#What's this?#

----------
MTSATのLRIT信号を受信するためのダウンコンバータ及びデコーダの回路図。Eagleで作成。

[Maker Faire Tokyo 2012](http://makezine.jp/)に出品予定。

こちらの記事に書いてあるものを基板に起こしたもの
